## ⚒️주요 기능
### **식단 기록**
- 식사 종류별 식단 기록 등록
  - 식품 데이터셋 기반의 식품 정보 검색
- 나만의 식품 정보 등록 
### **섭취 정보**
-  식단 기록의 음식 영양소를 기반으로 일일/주간/월간/연간 섭취량 조회
### **채식 레시피**
- 사용자의 채식 타입에 맞는 레시피 추천
- 레시피 등록/조회/수정/삭제
- 레시피 스크랩
### **커뮤니티**
- 게시글 등록/조회/수정/삭제
- 댓글/답글 작성
- 댓글/게시글 좋아요
### **알림**
- 커뮤니티 활동 관련 알림
- 권장 섭취량 초과 알림
